Guia Visual de Tags HTML

Esta página demonstra algumas das principais tags HTML e como elas aparecem no navegador.

1. Títulos

As tags de título são usadas para organizar o conteúdo e criar hierarquia.

Título grande (h1)

Título médio (h2)

Título menor (h3)

2. Parágrafo

A tag <p> é usada para escrever textos e organizar ideias.

Este é um parágrafo de exemplo.

3. Negrito e Destaque

<b> deixa o texto em negrito apenas visualmente.

Eu gosto de programação.

<strong> indica que o texto é importante.

Atenção: salve seu arquivo.

4. Itálico e Ênfase

<i> deixa o texto em itálico visual.

Estou estudando programação.

<em> indica ênfase no texto.

Aprender HTML é muito importante.

5. Quebra de Linha

A tag <br> quebra a linha sem criar um novo parágrafo.

Olá
mundo

A tag <hr> cria uma linha que separa conteúdos.
